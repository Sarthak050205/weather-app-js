# Weather App (JavaScript)

A browser-based weather application that fetches live weather data using the OpenWeather API.

## Features
- Search weather by city
- Displays temperature, humidity, description
- Real sky background UI
- Responsive card layout
- Async API handling with fetch

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## What I Learned
- Async/Await API calls
- DOM manipulation
- Error handling
- UI layering with CSS
